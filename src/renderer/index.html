AYUDA
Requisitos y configuración
Requisitos
1. Python 3.8+ en el PATH
2. ffmpeg en el PATH
3. Dependencias Python:pip install pydub requests google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client mistralai
Google Drive — primera vez
1. Ve a console.cloud.google.com
2. Crea un proyecto → activa la Google Drive API
3. Credenciales → Crear → ID OAuth → App de escritorio
4. Descarga el JSON → renómbralo credentials.json
5. Muévelo a ~/.srt_automatico/credentials.json
6. La primera vez se abre el navegador para autorizar — solo esa vez
Flujo recomendado
Procesar: Audio → FLAC → Voxtral → SRT en carpeta
